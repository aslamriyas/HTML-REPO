Introduction of Grouping Selector

  • This chapter teaches you how to group CSS selectors the right way, allowing you to declare the same styling for more than one element using one declaration block.
  • Together with their corresponding declaration block, the grouped selectors are called a CSS ruleset or simply a CSS rule.
  • You can place any valid selector in a group, and all elements in the document that match all the grouped elements will have the same style based on that style property.
